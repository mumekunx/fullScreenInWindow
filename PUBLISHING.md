# 公開・ストア申請チェックリスト

## 掲載時の単一目的

ユーザーが明示的に有効化した現在のタブで、サイトのフルスクリーン操作をOS全画面ではなくブラウザの表示領域内へ切り替えます。

## 掲載説明に含める事項

- 現在のタブでユーザーがONにした場合だけFull Screen APIを一時的に置き換える
- OFF、ページの再読み込み、ページ移動で変更を解除する
- サイトの構造によっては利用できない
- 閲覧履歴、ページ内容、動画、アカウント情報を収集・保存・送信しない
- 動画保存、DRM回避、広告の非表示・スキップを行わない
- 特定の動画配信サービスによる公認・提携製品ではない
- ウェブサイトの利用規約に従い、表示変更が許可されているページでのみ使用する

## 権限の申告文

### activeTab

ユーザーが拡張機能アイコンを操作した現在のタブに限り、フルスクリーン操作と表示を変更するために使用します。別のタブや閲覧履歴にはアクセスしません。

### scripting

拡張機能に同梱したJavaScriptとCSSを、ユーザーが明示的に有効化した現在のタブへ挿入するために使用します。外部コードは読み込みません。

## 公開前に設定・確認する項目

- [ ] 開発者名と問い合わせ先を確定する
- [ ] GitHub上の`PRIVACY.md`を公開し、ストアのプライバシーポリシーURLへ設定する
- [ ] ストアのデータ取り扱い申告を実装と一致させる
- [ ] Chrome、Edge、FirefoxでON、OFF、フルスクリーンボタン、`Esc`、再読み込みを確認する
- [ ] OFF後にネイティブのFull Screen APIへ戻ることを確認する
- [ ] ブラウザ内部ページで分かりやすいエラーになることを確認する
- [ ] 対象として宣伝する各サイトの最新利用規約を再確認する
- [ ] サービス名、ロゴ、スクリーンショットを掲載する場合は商標・許諾条件を確認する
- [ ] 配布ZIPに開発メモ、Git情報、不要ファイル、秘密情報が含まれていないことを確認する

## 使用を避ける表現

- 「すべてのサイトに完全対応」
- 「規約違反にならない」
- 「公式」「公認」「提携」など、事実と異なる関係を示す表現
- 許可を得ていないサービス名を並べた検索キーワード目的の説明
