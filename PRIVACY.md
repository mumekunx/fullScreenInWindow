# プライバシーポリシー / Privacy Policy

最終更新日: 2026年7月17日

## 日本語

Window Fullscreenは、個人情報、閲覧履歴、閲覧ページの内容、動画・音声、Cookie、認証情報、利用状況を収集、保存、販売、共有、外部送信しません。

拡張機能は、ユーザーがツールバーのアイコンをクリックし、現在のタブで明示的にONにした場合だけ、そのタブのFull Screen APIを一時的に置き換え、フルスクリーン対象要素の表示を端末内で変更します。対象要素やページ情報は保存・送信されません。OFF、ページの再読み込み、またはページ移動により変更は終了します。

使用する権限:

- `activeTab`: ユーザーが操作した現在のタブへ一時的にアクセスするため
- `scripting`: 拡張機能に同梱された表示用コードとCSSを現在のタブへ挿入するため

本ポリシーに関する問い合わせは、[GitHub Issues](https://github.com/mumekunx/fullScreenInWindow/issues)から連絡してください。個人情報や機密情報は投稿しないでください。

## English

Window Fullscreen does not collect, store, sell, share, or transmit personal information, browsing history, page content, video or audio, cookies, authentication information, or usage data.

The extension temporarily replaces the Fullscreen API and changes the display of the requested fullscreen element locally only after the user clicks the toolbar icon and explicitly enables the feature for the current tab. Target elements and page information are not stored or transmitted. The changes end when the feature is disabled, the page is reloaded, or the tab navigates away.

Permissions used:

- `activeTab`: Provides temporary access to the current tab after an explicit user action.
- `scripting`: Inserts the extension's packaged display code and CSS into the current tab.

For questions about this policy, contact the developer through [GitHub Issues](https://github.com/mumekunx/fullScreenInWindow/issues). Do not post personal or confidential information.
