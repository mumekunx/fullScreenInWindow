# Window Fullscreen

動画のフルスクリーンボタンを押したとき、画面全体（OS レベル）ではなく**ブラウザウィンドウいっぱい**に表示するブラウザ拡張機能。

YouTube・Netflix・Twitch など、あらゆるサイトの `<video>` 要素に対応しています。

---

## インストール方法

### Chrome / Edge（Chromium 系）

1. このリポジトリをクローンまたは ZIP でダウンロード・解凍する
2. Chrome で `chrome://extensions` を開く
3. 右上の **「デベロッパーモード」** トグルを **ON** にする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. ダウンロードしたフォルダ（`fullScreenInWindow`）を選択する
6. ツールバーにアイコンが表示されれば完了！

> **Edge の場合：** `edge://extensions` を開いて同じ手順で行う

### Firefox

1. このリポジトリをクローンまたは ZIP でダウンロード・解凍する
2. Firefox で `about:debugging` を開く
3. 左メニューの **「この Firefox」** をクリック
4. **「一時的なアドオンを読み込む...」** をクリック
5. フォルダ内の **`manifest.json`** を選択する
6. アドオンが読み込まれれば完了！

> **注意：** Firefox での一時的な読み込みはブラウザを閉じるとリセットされます。永続的に使うには AMO (addons.mozilla.org) への署名が必要です。

## 使い方

1. 動画が再生されているページを開く
2. プレイヤーのフルスクリーンボタンを押す（または `F` キーなどのショートカット）
3. 通常のフルスクリーンの代わりに、**ブラウザウィンドウいっぱい**に動画が表示される
4. 終了するには **`Esc` キー** を押すか、画面上部に表示される **「終了」ボタン** をクリックする

### コントロール

| 操作 | 説明 |
|------|------|
| `Esc` | ウィンドウフルスクリーンを終了 |
| マウスを動かす | 上部の「終了」ボタンを表示（3秒後に自動非表示） |
| 拡張機能アイコンをクリック | ON / OFF の切り替え |

---

## 拡張機能の有効 / 無効

ツールバーの拡張機能アイコンをクリックするとポップアップが開きます。  
右上のトグルスイッチで、ウィンドウフルスクリーン機能を ON / OFF できます。

- **ON（デフォルト）：** フルスクリーンボタンを乗っ取り、ウィンドウ内に表示
- **OFF：** 通常のフルスクリーン（OS 全画面）に戻る

---

## 仕組み

| ファイル | 役割 |
|---------|------|
| `manifest.json` | 拡張機能の設定（Chrome MV3 / Firefox 対応） |
| `injected.js` | ページのコンテキストで動作。`Element.requestFullscreen` を上書きして OS フルスクリーンをブロックし、カスタムイベントを発火 |
| `content.js` | `injected.js` を注入し、CSS の適用・オーバーレイ UI の管理・Esc キー処理を担当 |
| `content.css` | `position: fixed` でプレイヤーをビューポート全体に広げるスタイル |
| `popup.html/css/js` | 拡張機能アイコンのポップアップ UI |
