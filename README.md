# Window Fullscreen

利用が許可されたHTML5動画ページで、動画プレイヤーのフルスクリーン操作をOS全画面ではなくブラウザの表示領域いっぱいに切り替えるブラウザ拡張機能です。

> [!IMPORTANT]
> この拡張機能は、特定の動画配信サービスによる公式・公認・提携製品ではありません。技術的に動作する場合でも、そのサイトでの利用が許可されていることを意味しません。利用者自身が各サイトの利用規約と適用法令を確認し、表示やFull Screen APIの変更が許可されているページでのみ使用してください。

## 対応範囲と利用条件

想定している利用先:

- 自分が所有または管理しているウェブサイト
- サイト運営者から拡張機能の利用許可を得ているページ
- ローカルのテストページ、デモページ、利用条件上の変更が認められたHTML5動画ページ

対象外としている利用先・用途:

- 規約でコード挿入、インターフェース変更、プレイヤー機能の変更を禁止しているサイト
- 有料・会員制・DRM保護された配信サービス（運営者から明示的な許可を得た場合を除く）
- 広告、著作権表示、アクセス制限、地域制限、コピー制限などを回避・非表示にする目的
- 公衆上映、再配信、録画、ダウンロードなど、通常の個人視聴を超える目的

この拡張機能は、サイトの利用規約を自動判定したり、禁止サイトを自動的に遮断したりしません。このリポジトリでは、特定の動画配信サービスを対応対象として掲げません。判断できない場合は使用しないでください。

## プライバシーと権限を抑えた動作

- インストール時に全ウェブサイトへの常時アクセス権限を要求しません
- 拡張機能のアイコンをクリックし、現在のタブでONにした場合だけ動作します
- OFF、ページの再読み込み、ページ移動によりサイト側APIの変更は解除されます
- 閲覧履歴、動画、アカウント情報などを収集・保存・外部送信しません
- 広告の非表示、DRMの回避、動画の保存・抽出は行いません
- 独自の操作バーや広告を覆うUIは追加しません

ONの間は、サイトの`Element.requestFullscreen`などを一時的に置き換え、`document.fullscreenElement`をウィンドウ内表示の対象要素として返します。この変更は現在のタブ内だけで行われ、OFFにすると元のAPIへ復元します。

詳細は[プライバシーポリシー](PRIVACY.md)を参照してください。ストア申請時の確認事項は[公開チェックリスト](PUBLISHING.md)にまとめています。

## インストール方法

### Chrome / Edge（Chromium系）

1. このリポジトリをクローン、またはZIPでダウンロード・解凍する
2. Chromeでは`chrome://extensions`、Edgeでは`edge://extensions`を開く
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする
5. このプロジェクトのフォルダを選択する

### Firefox

1. `about:debugging`を開く
2. 「このFirefox」を選択する
3. 「一時的なアドオンを読み込む」をクリックする
4. このプロジェクトの`manifest.json`を選択する

Firefoxで一時的に読み込んだ拡張機能は、ブラウザを終了すると削除されます。一般配布にはAMOでの署名が必要です。

## 使い方

1. 利用規約またはサイト運営者の許可により、表示変更が認められている動画ページを開く
2. ツールバーの「Window Fullscreen」アイコンをクリックする
3. ポップアップのスイッチをONにする
4. 動画プレイヤーのフルスクリーンボタン、またはサイトの対応ショートカットを使用する
5. 終了するには`Esc`を押すか、ポップアップのスイッチをOFFにする

サイトの実装によっては正しく表示できない場合があります。表示中に広告、著作権表示、サイト標準の操作機能などが隠れる場合は、直ちにOFFにして使用を中止してください。特定サービスでの動作や継続的な互換性は保証しません。

## 権限

| 権限 | 用途 |
|---|---|
| `activeTab` | ユーザーが拡張機能を操作した現在のタブだけに、一時的にアクセスするため |
| `scripting` | 現在のタブへ同梱済みの表示用CSSとローカルスクリプトを挿入するため |

全サイトへの常時アクセス権限、閲覧履歴、ダウンロード、Cookie、アカウント情報への権限は要求しません。

## 仕組み

| ファイル | 役割 |
|---|---|
| `manifest.json` | Manifest V3設定と最小限の権限を定義 |
| `popup.html` / `popup.css` / `popup.js` | 現在のタブ単位でON/OFFするUI |
| `content.js` | ページ側スクリプトとの連携、表示状態、`Esc`操作を管理 |
| `injected.js` | ONの間だけFull Screen APIを置き換え、OFF時に元へ復元 |
| `content.css` | 対象プレイヤーをブラウザ表示領域へ広げるスタイル |
| `PRIVACY.md` | データの取り扱いと権限の説明 |
| `PUBLISHING.md` | ストア掲載文、権限理由、公開前チェックリスト |

## 利用上の注意

この拡張機能はウェブページのローカル表示とフルスクリーン機能を一時的に変更します。ウェブサイトによっては、利用規約でインターフェースやサービス機能の変更を制限しています。拡張機能ストアでの公開・承認、このリポジトリでの公開、技術的な動作確認のいずれも、各ウェブサイトから利用許諾を得たことを意味しません。

本プロジェクトは、特定サービスでの動作、利用規約への適合性、継続的な互換性を保証しません。利用規約に適合するか判断できない場合は、サイト運営者または法律の専門家へ確認してください。
